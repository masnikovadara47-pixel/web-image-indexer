# Описание
Это консольная программа на Python, которая автоматически скачивает все изображения с указанного сайта и сохраняет ссылки на них в текстовый файл.
Ручной сбор картинок с сайтов — долгий и трудоёмкий процесс. Программа делает это за секунды: пользователю достаточно ввести адрес страницы, а остальное инструмент берёт на себя. Особенно актуально для дизайнеров (сбор референсов), маркетологов (анализ конкурентов) и разработчиков (создание датасетов для нейросетей).

Запуск программы

2. Запустите скрипт:

3. Введите URL сайта, который хотите проиндексировать:
Введите URL веб-сайта для индексации: https://example.com
