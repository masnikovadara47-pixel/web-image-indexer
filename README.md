# Описание
Web Image Indexer — это современное десктопное приложение с графическим интерфейсом на Python (Tkinter) для автоматического сбора и загрузки изображений с веб-страниц. Программа анализирует указанный URL, извлекает все ссылки и изображения, а затем позволяет сохранить результаты одним кликом. Ручной сбор картинок с сайтов — долгий и трудоёмкий процесс. Программа делает это за секунды.
## Установка
Инструкция для тех, кто хочет запустить проект у себя.

Шаг 1: Установка Python

Перейдите на официальный сайт: https://python.org/downloads/
Нажмите кнопку «Download Python 3.x» (актуальная версия 3.7 или выше)
Запустите скачанный файл установки
ВАЖНО: Обязательно отметьте галочку «Add Python to PATH» внизу окна установки
Нажмите «Install Now»
Дождитесь завершения установки

Шаг 2: Установка необходимых библиотек

В командной строке/терминале по очереди выполните команды:

bash
pip install requests
pip install beautifulsoup4

Шаг 3: Скачивание программы с GitHub

Откройте браузер и перейдите по ссылке на репозиторий GitHub
Нажмите зелёную кнопку «Code» (вверху справа)
В выпадающем меню выберите «Download ZIP»
Распакуйте скачанный ZIP-архив в любую папку (например, на рабочий стол)

Шаг 4: Открытие программы в Python
Откройте папку, в которую распаковали архив
Найдите файл с названием web.ipynb
Дважды кликните левой кнопкой мыши по файлу web.ipynb
Откроется графическое окно программы

Шаг 5: Работа с программой

Введите URL сайта
Нажмите «Начать индексацию»
Для сохранения — «Сохранить ссылки» или «Скачать изображения»


Пример использования
Сбор изображений с новостного сайта

1. Запускаем программу

2. Вводим URL: https://news.ycombinator.com

3. Нажимаем "Начать индексацию"

4. Получаем статистику:
Найдено ссылок: 285
Найдено изображений: 15
Форматы: .png(12), .svg(3)

5. Нажимаем "Скачать изображения" — все логотипы и иконки сайта сохраняются в папку
