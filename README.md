# Описание
Web Image Indexer — это современное десктопное приложение с графическим интерфейсом на Python (Tkinter) для автоматического сбора и загрузки изображений с веб-страниц. Программа анализирует указанный URL, извлекает все ссылки и изображения, а затем позволяет сохранить результаты одним кликом. Ручной сбор картинок с сайтов — долгий и трудоёмкий процесс. Программа делает это за секунды.
## Установка
Инструкция для тех, кто хочет запустить проект у себя.

git clone https://github.com/ваш-username/web-image-indexer.git
cd web-image-indexer
pip install requests beautifulsoup4 # Установка зависимостей
python web_indexer_gui.py # запуск



## Использование
Запуск программы
1. Запустите программу — откроется главное окно с тёмным интерфейсом

2. Введите URL страницы для анализа (например, https://example.com)

3. Нажмите "Начать индексацию" — программа проанализирует страницу

4. Изучите результаты:
    Статистика покажет количество найденных ссылок и изображений
    В журнале отобразятся все форматы изображений

5. Сохраните данные:
    "Сохранить ссылки" — все ссылки в текстовый файл
    "Скачать изображения" — загрузка всех картинок в папку

6. При необходимости смените папку сохранения через кнопку "Папка сохранения"


Пример использования
Сбор изображений с новостного сайта

1. Запускаем программу

2. Вводим URL: https://news.ycombinator.com

3. Нажимаем "Начать индексацию"

4. Получаем статистику:
Найдено ссылок: 285
Найдено изображений: 15
Форматы: .png(12), .svg(3)

5. Нажимаем "Скачать изображения" — все логотипы и иконки сайта сохраняются в папку
